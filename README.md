# Usage
<!-- usage -->
```sh-session
$ npm install -g fph
$ fph COMMAND
running command...
$ fph (-v|--version|version)
fph/1.0.0 darwin-x64 node-v10.15.0
$ fph --help [COMMAND]
USAGE
  $ fph COMMAND
...
```
